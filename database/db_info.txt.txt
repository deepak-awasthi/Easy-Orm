Our application by default uses easy_orm database as the name,localhost as the host,root as the username and ''(empty) as the password
.If you would like to change these details with your own you can edit the details in the file db_constant.php.You can edit the protected properties of that class. 
and it has a table called tbl_blog.You can create the table structure with fields and datas with easy_orm.sql file in  the same directory where this file is located.
